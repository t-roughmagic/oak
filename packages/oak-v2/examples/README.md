# Oak v2 Examples

Scratchpad examples for the v2 platform-owned program model.

## `react-effect-commands`

End-to-end Effect-platform command demo:

- create a standalone Oak Effect program
- provide its `Layer` through `@oak/react-effect-provider`
- resolve the running service and pass its view driver to Oak's React adapter
- dispatch an Effect command from a React button and render the resulting state
